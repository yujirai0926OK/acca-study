# ACCA FA 対訳リーダー＆単語帳

ACCA Financial Accounting (FA) 学習用の、単一HTMLアプリ。
ブラウザだけで動作（サーバー不要・オフライン可・無料・API課金なし）。

## 機能
- **📖 リーダー**：第1章 各セクションの英日対訳の要点。「マイ対訳」欄に自分の本文を貼り付けて保存可（端末内）。
- **🗂 単語帳**：第1章キー用語 55語を収録。Leitner方式の間隔反復で暗記。単語の追加・削除可。
- **✍️ テスト**：第1章の概念から作成した選択問題8問。即時採点＋解説。
- **⚙️ データ**：進捗・単語の書き出し／読み込み（JSON）。PC↔iPadの同期・バックアップに。

## iPad / iPhone での使い方
1. `index.html` をOneDriveに置く → ファイルアプリ／ブラウザで開く
2. Safariで開き、共有 →「ホーム画面に追加」でアプリ化（オフライン可・進捗が消えにくい）

## 注意
- 進捗はブラウザのlocalStorageに保存。長期間開かないと消えることがあるため、ときどき「書き出し」でバックアップする。
- 教材本文（Kaplan著作物）は収録していない。本文は「マイ対訳」欄に各自で貼り付けて個人学習用に使うこと。配布・公開はしない。
